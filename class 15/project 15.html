explain container

The class="container" in HTML is a Bootstrap class that centers and gives a fixed width to its content, making it responsive across different screen sizes. It helps in maintaining consistent alignment and padding around the content within the container

explain fluid container

The class="container-fluid" in Bootstrap creates a full-width container that spans the entire width of the viewport, regardless of screen size. It is used when you want the content inside the container to stretch across the entire width of the page.

container border

The class="container border" in Bootstrap creates a centered container with a fixed width and adds a border around it. The container class manages the layout and centering, while the border class adds a 1px solid border around the container, providing a visual outline.
